# markdownの書き方

## 編集のしかた（3ステップ）

1. ブラウザでこのリポジトリを開き、`docs/` から自分の章のファイルをクリック
2. 右上の鉛筆アイコン（Edit）を押して書く。「Preview」タブで見た目を確認できる
3. 右上の「Commit changes...」→ そのまま「Commit changes」（mainに直接コミットでOK）

## 基本の記法

普通の文章はそのまま書けばOK。段落を分けたいときは空行を1行入れる。

### 見出し

```
# 大見出し（章タイトル）
## 中見出し（節タイトル）
### 小見出し
```

### 箇条書き

```
- 項目1
- 項目2
```

番号付きにしたいとき:

```
1. 項目1
2. 項目2
```

### 強調

```
**太字** になる
```

### リンク

```
[表示したい文字](https://example.com)
```

URLをそのまま貼ってもリンクになる。

### 画像

```
![説明](画像のURLやパス)
```

編集画面に画像ファイルをドラッグ&ドロップすると、この形が自動で入る。
