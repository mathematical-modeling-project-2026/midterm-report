# 第1章 はじめに

本章では，本プロジェクトが対象とする問題の背景，解決すべき課題および到達目標について述べる．

## 1.1 背景

公立はこだて未来大学には，函館駅前，五稜郭および亀田支所前方面と大学との間を結ぶ路線バスが運行されており，学生の通学を支える交通手段の一つとなっている．

本プロジェクトでは，身近な問題について検討する中で，朝や夕方の通学時間帯に，未来大学を発着する一部の便が混雑していることに着目した．利用者が特定の便に集中した場合，車内の快適性が低下するほか，車両の定員を上回る需要によって，希望する便に乗車できない利用者が生じる可能性がある．また，停留所での乗降に時間を要することで，運行の遅延や定時性の低下につながる可能性もある．

混雑に対する直接的な解決策として増便が考えられるが，増便には新たな車両や運転手の確保，運行コストの増加などが伴う．函館市においても，公共交通を担う運転手の不足が課題として挙げられている．そのため，新たな車両や運転手の追加だけに頼らず，既存の運行資源を活用しながら混雑を緩和する方法を検討する必要がある．

## 1.2 課題

現時点では，未来大学を発着するバスについて，時間帯や便ごとの乗車人数，混雑度，乗車できなかった利用者数および遅延時間などを定量的に把握できていない．そのため，登下校時間帯における混雑の実態を調査し，どの区間や時間帯に問題が生じているかを明らかにする必要がある．

また，バスの運行方法を変更した場合には，学生だけでなく，途中の停留所を利用する一般利用者やバス事業者にも影響が生じる可能性がある．しかし，実際の路線で複数の運行方法を試験的に実施することは，日常的な利用者への影響が大きいため困難である．

したがって，現在の運行状況と新たな運行方法を仮想的に比較し，混雑や遅延，利用者およびバス事業者への影響を定量的に評価することが課題となる．

## 1.3 到達目標

本プロジェクトでは，数理モデリングを用いて，未来大学を発着するバスの運行と利用者の移動を表現するモデルを構築する．そのモデルを用いて，現在の運行と複数の運行方法をシミュレーション上で比較し，混雑や遅延を緩和できる運行条件を検討する．

最終的には，学生，一般利用者およびバス事業者への影響をそれぞれ評価し，各主体の間に生じる利益と不利益を整理したうえで，既存の運行資源を活用した実現可能な運行方法を提案することを目指す．
