# 第1章 はじめに

本章では，本プロジェクトが対象とする問題の背景，解決すべき課題，および目的とアプローチについて述べる．

## 1.1 背景

公共交通機関の混雑は，多くの都市に共通する課題である．特に通勤・通学の時間帯には利用者が特定の便に集中しやすく，快適性の低下や遅延，乗車できない利用者の発生など，日常生活に直結する問題を引き起こす．一方で，地方都市では運転手の不足などにより輸送力の増強が容易ではなく，限られた運行資源の中で混雑に対処することが求められている．函館市においても，公共交通を担う運転手の不足が課題として挙げられている．

公立はこだて未来大学（以下，未来大学）には，函館駅前，五稜郭および亀田支所前方面と大学との間を結ぶ路線バスが運行されており，学生の通学を支える交通手段の一つとなっている．

本プロジェクトでは，身近な問題について検討する中で，朝や夕方の登下校時間帯に，未来大学を発着する一部の便が混雑していることに着目した．利用者が特定の便に集中した場合，車内の快適性が低下するほか，車両の定員を上回る需要によって，希望する便に乗車できない利用者が生じる可能性がある．また，停留所での乗降に時間を要することで，運行の遅延や定時性の低下につながる可能性もある．

（文責　小山内漣）

## 1.2 課題

混雑に対する直接的な解決策として増便が考えられるが，増便には新たな車両や運転手の確保，運行コストの増加などが伴う．前述の通り，函館市では公共交通を担う運転手の不足が課題となっており，増便だけに頼った解決は現実的ではない．そのため，既存の運行資源を活用しながら混雑を緩和する方法が求められる．

しかし，現時点では，未来大学を発着するバスについて，時間帯や便ごとの乗車人数，混雑度，乗車できなかった利用者数および遅延時間などが定量的に把握できていない．

また，バスの運行方法を変更した場合には，学生だけでなく，途中の停留所を利用する一般利用者やバス事業者にも影響が生じる可能性がある．一方で，実際の路線で複数の運行方法を試験的に実施することは，日常的な利用者への影響が大きいため困難である．

以上より，本プロジェクトの課題は次の2点である．1つ目は，登下校時間帯における混雑の実態を定量的に明らかにすることである．2つ目は，現在の運行状況と新たな運行方法を仮想的に比較し，混雑や遅延，利用者およびバス事業者への影響を定量的に評価することである．

（文責　小山内漣）

## 1.3 目的とアプローチ

本プロジェクトの目的は，既存の運行資源を活用しながら，混雑や遅延の緩和が期待でき，かつ実現可能なバスの運行方法を提案することである．あわせて，運行条件が学生，一般利用者およびバス事業者に与える影響を整理し，各主体の間に生じるトレードオフを明らかにする．

この目的を達成するためのアプローチとして，未来大学を発着するバスの現在の運行状況を，検証に必要な範囲で再現できる数理モデルおよびシミュレーションを構築する．モデルの構築にあたっては，バスの運行および混雑に関係するデータを収集し，必要な条件や仮定を設定する．そのうえで，構築したモデルを用いて現在の運行と複数の運行パターンを比較し，それぞれが混雑や遅延に与える影響を定量的に評価する．

（文責　佐々木出）
