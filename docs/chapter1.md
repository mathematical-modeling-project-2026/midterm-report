# 第1章 はじめに

本章では，本プロジェクトが対象とする問題の背景，解決すべき課題および到達目標について述べる．

## 1.1 背景

公立はこだて未来大学には，函館駅前，五稜郭および亀田支所前方面から路線バスを利用してアクセスでき、路線バスは学生の通学を支える交通手段の一つとなっている．
本プロジェクトでは，身近な問題について検討する中で，朝や夕方の通学時間帯に、未来大学を発着する一部の便が混雑していることに着目した．利用者が特定の便に集中した場合，車内の快適性が低下するだけでなく，停留所での乗降に時間を要することで，運行の遅延や定時性の低下につながる可能性がある．また，バスの定員を超える需要が生じた場合には，利用者が希望する便に乗車できない可能性もある．ただし，現時点では未来大学方面のバスにおける混雑状況や遅延の発生状況を定量的に確認できていないため，今後の調査によって実態を明らかにする必要がある．
一方，混雑に対する直接的な解決策として増便が考えられるが，増便には新たな車両や運転手の確保，運行コストの増加などが伴う．函館市は，労働時間規制の強化や運転手不足の影響を受け，路線バスの減便や運行内容の見直しが必要な状況にあるとしている．そのため，新たな車両や運転手の追加を前提とせず，既存の運行資源を活用して輸送効率を向上させる方法を検討する必要がある．
そこで本プロジェクトでは，一部の便を主要な停留所のみに停車する快速便として運行し，快速化によって短縮された時間を利用して，混雑が想定される亀田支所前から未来大学までの区間を同一車両で2周する案を検討対象とした．この運行案は，新たな車両や運転手を追加することなく，混雑区間における輸送回数を増やすことを目的とするものである．ただし，快速化によって実際に2周目の運行に必要な時間を確保できるか，また，混雑や遅延をどの程度改善できるかは明らかになっていない．したがって，本プロジェクトでは，この運行案の実現可能性と導入効果を数理モデルおよびシミュレーションによって検証する．

## 1.2 課題

前節で述べた快速化と2周運行を組み合わせた運行案については，現時点では，その実現可能性および利用者やバス事業者に与える影響が明らかになっていない．快速便が停車する停留所や対象とする便によって所要時間の短縮量は異なり，短縮された時間だけで亀田支所前から未来大学までを再度運行できるとは限らない．また，快速便を設定しても利用者が想定どおりに複数の便へ分散するとは限らず，特定の便や停留所に新たな混雑が生じる可能性もある．さらに，快速便が通過する停留所の利用者にとっては，利用できる便が減少し，待ち時間が増加する可能性がある．

しかし，実際の路線において，既存の便を快速化するなどの運行変更を試験的に実施することは，日常的にバスを利用する学生や地域住民に影響を与えるため，実環境で複数の運行条件を繰り返し検証することは困難である．

したがって，本プロジェクトにおける課題は，現在の運行と提案する運行案を比較できる数理モデルおよびシミュレーション環境を構築し，快速化と2周運行の実現可能性および導入効果を定量的に評価することである．モデル化にあたっては，現在の時刻表，停留所間の所要時間，停留所ごとの乗降需要，乗降に要する時間，車両の定員など，バスの運行と混雑に関係する要素を扱う．利用者による普通便と快速便の選択については，必要なデータを収集したうえで，モデルの目的に応じた仮定を設定する．

構築したモデルを用いて，快速化する便，快速便が停車する停留所，2周目の運行を開始する時刻などの条件を変更し，現在の運行と提案する運行案を比較する．これにより，混雑や遅延だけでなく，乗車できない利用者，一般利用者の待ち時間およびバス事業者の運行への影響を評価できるようにすることが課題となる．

## 1.3 到達目標

上記の課題に対し，本プロジェクトでは，現在のバス運行を検証に必要な範囲で再現できる数理モデルおよびシミュレーションを構築する．そのうえで，一部の便を快速化し，短縮された時間を利用して亀田支所前から未来大学までの区間を2周する複数の運行パターンを設定し，現在の運行との比較を行う．

評価項目としては，便ごとの乗車人数および乗車率，車内の混雑度，乗車できない利用者数，利用者の待ち時間，未来大学への到着時刻，運行の遅延などを想定する．また，快速便が通過する停留所の利用者に生じる待ち時間や乗車機会の変化についても評価する．バス事業者への影響については，取得可能なデータの範囲で，運行時間，輸送人数，運賃収入および運行コストなどの変化を評価する．

本プロジェクトの最終的な到達目標は，学生，一般利用者およびバス事業者のすべてに利益をもたらすことをあらかじめ前提とするのではなく，運行条件によって各主体にどのような利益または不利益が生じるかを定量的に明らかにすることである．そのうえで，各主体の間に生じるトレードオフを整理し，特定の主体に過度な不利益を生じさせず，現状と比較して全体的な改善が期待できる実現可能な運行条件を提示することを目指す．

具体的には，快速便の導入によって，学生のピーク時間帯における車内混雑や通学時間が改善するかを評価する．また，学生の利用が普通便と快速便に分散することで，途中停留所から乗車する一般利用者の乗車機会や車内の混雑状況がどのように変化するかを評価する．さらに，乗降時間や運行時間の変化が定時性に与える影響を検証するとともに，輸送人数，運賃収入および運行コストへの影響を評価する．これらの結果を総合的に比較し，混雑緩和と実現可能性の両面から望ましい運行パターンの条件を明らかにする．
