# 第1章 はじめに

本章では，本プロジェクトが対象とする問題の背景，解決すべき課題および到達目標について述べる．

## 1.1 背景

公立はこだて未来大学には，函館駅前，五稜郭および亀田支所前方面と大学との間を結ぶ路線バスが運行されており，学生の通学を支える交通手段の一つとなっている［1］．

本プロジェクトでは，身近な問題について検討する中で，朝や夕方の登下校時間帯に，未来大学を発着する一部の便が混雑していることに着目した．利用者が特定の便に集中した場合，車内の快適性が低下するほか，停留所での乗降に時間を要することで，運行の遅延や定時性の低下につながる可能性がある．また，車両の定員を上回る需要が生じた場合には，利用者が希望する便に乗車できない可能性もある．ただし，現時点では，未来大学を発着するバスの混雑状況や遅延の発生状況を定量的に確認できていないため，今後の調査によって実態を明らかにする必要がある．

混雑に対する直接的な解決策として増便が考えられるが，増便には新たな車両や運転手の確保，運行コストの増加などが伴う．函館市においても，公共交通を担う運転手の不足が課題として挙げられている［2］．そのため，新たな車両や運転手の追加を前提とせず，既存の運行資源を活用して輸送効率を向上させる方法を検討する必要がある．

そこで本プロジェクトでは，一部の便を主要な停留所のみに停車する快速便とし，快速化によって短縮された時間を利用して，混雑が想定される亀田支所前から未来大学までの区間を同一車両で2周する運行案を検討対象とした．この運行案は，新たな車両や運転手を追加せずに，混雑区間における輸送回数を増やすことを目的とするものである．

## 1.2 課題

前節で述べた運行案については，快速化によって2周目の運行に必要な時間を確保できるか，また，利用者やバス事業者にどのような影響を与えるかが明らかになっていない．快速化による所要時間の短縮量は，対象とする便や停車する停留所によって異なる．また，快速便を導入しても，利用者が普通便と快速便に想定どおり分散するとは限らず，特定の便や停留所に新たな混雑が生じる可能性がある．さらに，快速便が通過する停留所の利用者については，利用できる便の減少や待ち時間の増加などの影響も考慮する必要がある．

一方，実際の路線で既存の便を快速化し，複数の運行条件を試験することは，日常的にバスを利用する学生や一般利用者に影響を与えるため困難である．したがって，現行の運行と提案する運行案を仮想的に比較し，快速化と2周運行の実現可能性および導入効果を定量的に評価できる環境が必要となる．

本プロジェクトでは，時刻表，停留所間の所要時間，時間帯や停留所ごとの乗降需要，車両の定員など，バスの運行と混雑に関係する要素を検証に必要な範囲でモデルに取り入れる．普通便と快速便への利用者の割り振りについては，収集可能なデータと設定した仮定に基づいて表現する．これにより，快速化する便や停車する停留所などの条件を変更し，混雑，遅延，乗車できない利用者および一般利用者への影響を比較できるようにすることが課題である．

## 1.3 到達目標

上記の課題に対し，本プロジェクトでは，現在のバス運行を検証に必要な範囲で再現できる数理モデルおよびシミュレーションを構築する．そのうえで，一部の便を快速化し，短縮された時間を利用して亀田支所前から未来大学までの区間を2周する複数の運行パターンを設定し，現行の運行との比較を行う．

主な評価項目として，便ごとの乗車人数および乗車率，車内の混雑度，乗車できない利用者数，待ち時間，目的地への到着時刻および遅延時間を想定する．また，快速便が通過する停留所の利用者について，待ち時間や乗車機会がどのように変化するかを評価する．バス事業者への影響については，取得可能なデータの範囲で，運行時間，輸送人数，運賃収入および運行コストなどの変化を評価する．

本プロジェクトの最終的な到達目標は，学生，一般利用者およびバス事業者のすべてに利益が生じることを前提とするのではなく，運行条件によって各主体にどのような利益または不利益が生じるかを定量的に明らかにすることである．そのうえで，各主体の間に生じるトレードオフを整理し，特定の主体に過度な不利益を生じさせず，現行の運行と比較して全体的な改善が期待できる実現可能な運行条件を提示することを目指す．
