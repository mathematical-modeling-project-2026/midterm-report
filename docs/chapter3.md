# 第3章  前期活動内容について

## 3.1 輪講会

## 3.1.1 輪講会実施の理由と目的

　本プロジェクトの根幹には，「数理モデリングを用いて，函館市や公立はこだて未来大学に関わる人々にとって有益な成果物を作成する」という目標がある．そのため，数理モデルの構築はプロジェクトの中核的要素であり，不可欠なものである．しかし，プロジェクト開始当初，メンバーの多くが数理モデルの構築経験を持っておらず，必要な理論や技術に対する理解が十分ではなかった．そこで私たちは，まず既存の数理モデルを題材にした輪講会を実施することで，基礎的な知見の獲得を図ることにした．また，数理モデルを構築する過程で，数値計算法に関する知識も求められる場面が多く，これを理解せずにモデルを構築することは困難であると判断した．たとえ最終的に使わない技術であっても，選択の判断を下すためには一定の学習が不可欠であると考え，学びを深めることを優先する姿勢で輪講会を進めることにした．
以上を踏まえ，輪講会の目的は以下の 2 点に集約される．1 つ目：プロジェクトメンバーが数理モデルの構築経験を積むとともに，数値計算法に関する基本的な知識を身につけること．2 つ目：輪講会で得た知識をもとに，プロジェクトテーマに沿った数理モデルを作成する際，必要な知識を自律的に取捨選択できるようになること．このように，輪講会はプロジェクトの基盤づくりとして重要な役割を担っていた．

## 3.1.2 輪講会で使用した書籍とその選定理由

　本プロジェクトの輪講会では，森北出版の『Python による数値計算法の基礎』[] と、ソシム株式会社の『データ分析のための数理モデル入門』を使用した．
　『Python による数値計算法の基礎』は、数値計算に必要な理論を平易に解説しつつ、Python による実装方法にも丁寧に触れており、数理モデリングに必要な数値解法の学習に適していると判断した．書籍の内容には，ニュートン法やオイラー法，補間法，常微分方程式の数値解法など，プロジェクトで扱うテーマに直接関わる技術が多く含まれている．特に，演習問題が豊富であった点が選定の決め手となった．実際に手を動かしてモデルを構築し，数値的な近似や解法を体験できることにより，理論の理解を深めるとともに実践的なスキルの習得にもつながった．また，Python を使用する構成になっていたことも，本書を選んだ理由のひとつである．私たちのプロジェクトではモデルの実装や検証に Python を用いており，学習と実務が直結する形でスムーズに接続できた．
　『データ分析のための数理モデル入門』は、さまざまなモデリング手法の基礎を解説するだけでなく、それらをどのように選択し活用すればよいかについても丁寧に説明されており、モデリング手法の学習に適していると判断した．書籍の内容には，回帰分析や分類，クラスタリング，ベイズ推定，モデル選択など，データ分析に必要な基本的な手法が幅広く含まれている．特に，単に手法を解説するだけでなく，目的に応じたモデルの選択方法や適用場面についても説明されていた点が選定の決め手となった．数理モデリングでは，課題に適したモデルを選択し，結果を適切に評価することが重要であるため，本書を通して各手法の特徴や使い分けを理解することができた．また，図を交えながら丁寧に説明されていたため，各手法の特徴や違いを理解しやすく，効率的に学習を進めることができた． 
　このように，本書は輪講会の目的に非常に合致しており，プロジェクト全体の基礎力の向上に大きく貢献した教材であった．


 ## 3.1.3 輪講会で学習した内容

　本プロジェクトの輪講会では，数理モデリングの基礎となる数値計算法や関連理論を体系的に学習した．以下に，各回の学習内容と取り組みの様子を詳述する．
　初回では，非線形方程式の解法として，はさみうち法と二分法という基礎的な数値計算手法を学んだ.課題として提示された「(f(a)) と (f(b)) が同符号の場合には適用できない」という制約を通して，数値計算手法には適用条件があり，それらを満たして初めて正しく解を求められることを理解した.また，実際のモデリングでは，対象となる関数の性質や初期値の設定によって計算結果が大きく左右されるため，手法を適切に選択することの重要性を学んだ.さらに，コードを実行することで，はさみうち法と二分法をPythonで実装する方法や，実装した際の動作について理解を深めることができた.
 　第 2 回では，数値積分の代表的手法である台形法とシンプソン法を扱った．台形法は，関数を区間ごとに直線（台形）で近似するため比較的計算が単純であり，手軽に実装できる利点がある．一方で，シンプソン法は区間を 2n 分割し，2 次関数で近似するため精度が高く．特に滑らかな関数に対して有効であることを学んだ．演習では，各自が Python でシンプソン法を実装し，偶数インデックスと奇数インデックスに異なる重みを与える公式に基づいた数式展開のコード化に取り組んだ．特に，分割数が偶数でないとうまくいかない点や，端点の扱い方で誤差が出やすいことを体験的に理解できた点が，実装の有意義な側面だった．
　第 3 回では，データの近似をテーマに，最小二乗法，ニュートンの補間法，スプライン補間法を取り上げた．最小二乗法では，誤差の 2 乗和を最小化するという観点から，実験値や観測値に最も「らしい」直線を当てはめる方法を学んだ．実際の観測データを用いて回帰直線を作成し，その結果を可視化することで，統計処理と数理的な構築の両面があることを体感した．ニュートンの補間法では，差分商を使って関数を多項式で表現する過程を学び，補間点が増えるほど精度は上がるが，過剰な次数の多項式は振動を起こしやすい（ランゲ現象）ことも議論された．スプライン補間では，特に 3 次関数を用いる理由として「2 次では滑らかさが足りず，4 次以上は過剰近似となることが多い」ため，3 次が最もバランスが良いとされている点が興味深かった．グラフを Pythonで描きながら，補間の滑らかさの違いを視覚的に確認したのは，プロジェクトでのシミュレーション描画にも通じる体験だった．
　第 4 回では，連立一次方程式を解く方法として，緩和法，掃き出し法，ガウス・ザイデル法を取り上げた．緩和法は問題の制約を弱めることで解きやすくする手法，掃き出し法は拡大係数行列に対して行基本変形を繰り返す「直接法」，ガウス・ザイデル法は逐次的に変数を更新する「反復法」としてそれぞれの分類と特徴を学んだ．特に，ガウス・ザイデル法において「対角優位行列である必要がある」などの条件が収束性に関わる点は重要な学びであり，実際に Python による第 2 近似までのコードを動かして結果を確認した．コードの各部分の動きについてチームで逐一確認しながら理解を深めることができ，実装面でも収穫が大きかった．
　第 5 回では，微分方程式の数値解法としてテイラー法，オイラー法，ルンゲクッタ法を学んだ．これらはいずれも連続的な変化を離散的に近似する方法であり，特に物理現象や人流などの時間発展を表現するために重要な技術である．オイラー法は実装が非常に簡単である一方，刻み幅が大きいと精度が悪化しやすい．ルンゲクッタ法はその精度を補う代表的な手法として位置づけられた．この回では，連立常微分方程式の解法や高階常微分方程式を扱い，変数を一次の系に変換する方法も学習した．Python で逐次計算を行い，数ステップ分の変化をグラフで確認することで，数値積分と現実世界のシミュレーションのつながりを意識できた．
　最終回では，偏微分方程式を数値的に扱うための差分法，時間領域差分法（FDTD 法），そして有限要素法（FEM）といった高度な手法を学習した．差分法では，空間や時間を細かく分割して近似することで，波動方程式などを解く手法を扱った．実際に波動方程式にガウスの定理やストークスの定理を適用し，電磁気学の基本法則（アンペールの法則、ファラデーの法則）との関係についても確認した．さらに，モーメント法やガラーキン法，最小二乗法，コロケーション法，部分領域法といった手法も概観し，それぞれの誤差の扱いや適用対象の違いについて学んだ．特にガラーキン法では「誤差が基底関数と直交するようにする」という概念が新鮮だった．有限要素法については，空間を小さな要素に分割し，各要素の数理的振る舞いを合成して全体の解を求める手法であり，実際の構造解析や流体解析にも広く用いられていることを知った．プロジェクトの応用として，混雑シミュレーションに空間的要素を導入する際に応用可能性があると考えられた．
　全体を通して，輪講会では理論の学習にとどまらず，Python での実装を通じて知識の定着と応用力の向上を図ることができた．また，毎回の学習内容はプロジェクトの目的である「現実の課題を数理的に捉える」ための土台となっており，輪講会で得た知識は今後のモデル構築・検証・改善フェーズにおいて中心的な役割を果たすことが期待される．
