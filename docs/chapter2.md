# 第2章 先行研究

本章では，本プロジェクトの提案と関連する先行研究の知見を整理し，それらに対する本プロジェクトの立ち位置を述べる．

（文責　山本貫太）

## 2.1 快速バス導入による効果

バス路線網の再編や運行形態の導入に関する研究は数多く行われている．中でも，本プロジェクトの提案と関連する「快速バスの導入」に着目した一連の研究として，数理最適化と社会実験の事例が挙げられる．

高山らは，快速バスを混在させた最適バス路線網計画の支援システムを提案している（高山ら，1998）．この研究では，快速バスの導入が有効な路線の条件として，「郊外起点側で多く乗車し，都心側でほとんどの乗客が下車するようなタイプ」を挙げており，途中の停留所での乗降が少ない区間を通過することによる所要時間短縮の効果を示している．さらに，運行効率の観点から，快速バスで運行することにより「バスの回転率が上がり，需要の大きな路線に，より多くのバスを配車することが可能となる」という，バス事業者側のメリットも指摘されている．

また，高山，塩土は，実際に金沢市において朝の通勤時間帯に快速バスを導入する社会実験を実施し，その事前・事後分析を行っている（高山・塩土，1996）．この実験では，7つの停留所を通過して所要時間を5分短縮する快速バスを運行し，沿線住民を対象とした調査を実施している．その結果，事前調査の段階では「運行頻度の増加」や「時間短縮」が高く評価され，約400人の利用転換が予測された．しかし，実際の利用者増は日平均92人にとどまっている．事後調査の分析により，アンケート等に基づく主観的な需要予測は楽観的になりやすく，実際の行動変容には乗車時間や停留所までの徒歩時間といった現実的な要因が影響することが示された．

これらの先行研究から，快速バスの導入は事業者と利用者の双方にメリットをもたらす手段である反面，単なる主観的な意向調査だけでは現実の利用実態との間に乖離が生じやすいことがわかる．したがって，導入にあたっては，満員で乗車できない利用者（以下，積み残し）の発生などの現実的な制約を考慮し，全体の所要時間を最適化するための運行計画が必要となる．

（文責　山本貫太）

## 2.2 本プロジェクトの立ち位置

先行研究で示された「快速化によるバス回転率の向上」という知見は，本プロジェクトが第4章で提案する「快速化によって短縮できた時間を利用し，同一車両で混雑区間をもう1周運行する」という解決策の裏付けとなる．

しかし，既存の研究は都市全体の広域なネットワークにおける路線網とダイヤの同時決定を目的としたものが中心である．本プロジェクトの対象である未来大学に接続する路線バスのように，「特定の時間帯」および「特定の区間」に需要が集中するという局所的なボトルネックの解消に特化した検証事例は少ない．さらに，本プロジェクトでは，運転手不足などを背景とした「新たな車両や人員の増強に頼らず，既存の運行資源を活用する」という制約も考慮する必要がある．

そこで本プロジェクトでは，広域な路線網全体の最適化ではなく，需要が集中する特定の区間・時間帯に対象を絞ったシミュレーションモデルを構築する．その際，先行研究で示された主観的予測の限界を踏まえ，意向調査に頼るのではなく，各停留所における乗客の到着や乗降，定員超過による積み残しの発生を数理モデルとして表現することで，運行方法の変更が及ぼす影響を客観的に評価できるようにする．このモデルを用いて，一部の便の快速化と2周運行の導入が，学生の利便性，途中の停留所を利用する一般利用者，バス事業者の運行効率に与える影響を定量的に評価し，実現可能性の高い運行案の提案につなげる．

（文責　山本貫太）
