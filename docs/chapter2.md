# 第2章 数理モデリングの概要

本章では，数理モデリングの定義と基本的な流れについて述べる．また，数理モデルとシミュレーションの関係を整理したうえで，本プロジェクトにおいて数理モデリングを用いる意義について述べる．

## 2.1 数理モデリングとは

数理モデリングとは，現実に生じている現象や問題から重要な要素を抽出し，それらの関係を数式や規則などによって表現することで，現象の理解，予測および評価を行うための手法である［1］．作成された数学的な表現を数理モデルという．

現実の現象には，多くの要素が複雑に関係している．そのすべてを数理モデルに取り入れることは困難であるため，数理モデリングでは，対象とする問題において重要と考えられる要素を選び，それ以外の要素を省略する．また，現実の状況を扱いやすい形に単純化するために，必要に応じて仮定を設定する．

したがって，数理モデルは現実をそのまま再現したものではなく，特定の目的に応じて現実の一部を表現したものである．同じ現象を対象とする場合でも，分析する目的や重視する要素が異なれば，構築される数理モデルも異なる．

## 2.2 数理モデリングの手順

数理モデリングは，一般に，現実の問題を設定し，必要な要素を抽象化したうえで数学的に表現し，得られた結果を現実の問題に照らして評価するという流れで行われる［2］．

はじめに，現実の現象から対象とする問題を明確にする．この段階では，何が問題となっているのか，何を明らかにしたいのか，どのような結果を得たいのかを整理する．

次に，対象とする現象の中から，問題に大きく関係すると考えられる要素を選択する．現実のすべての要素を扱うことは困難であるため，必要な仮定を設定し，現象を簡略化する．そのうえで，対象となる要素を変数として定め，要素間の関係を数式，確率，規則またはアルゴリズムなどによって表現する．

構築した数理モデルは，数学的な解析やコンピュータによる数値計算などを用いて調べる．得られた結果を解釈し，現実の現象や観測データと比較することで，モデルが目的に対して適切であるかを評価する．

モデルによる結果と現実の状況に大きな差がある場合には，設定した仮定，使用したデータ，変数間の関係などを見直す必要がある．このように，数理モデリングは一方向に進むものではなく，結果の評価に基づいて前の段階へ戻り，モデルを修正する過程を繰り返すものである．

数理モデリングの基本的な流れを以下に示す．

1. 対象とする現象および問題を設定する
2. 重要な要素を抽出し，仮定を設定する
3. 変数および要素間の関係を数学的に表現する
4. 数理モデルを解析し，必要に応じてシミュレーションを行う
5. 結果を解釈し，現実のデータと比較する
6. 仮定やモデルの構造を評価し，必要に応じて修正する
7. 得られた結果を問題の理解や解決策の検討に用いる

## 2.3 数理モデルとシミュレーションの関係

数理モデルとシミュレーションは密接に関係しているが，同一のものではない．数理モデルは，現実の現象を数学的に表現したものである．一方，シミュレーションは，構築したモデルに条件や数値を与え，時間の経過や条件の変化によって結果がどのように変わるかを計算によって調べる方法である．

数理モデルを調べる方法には，数式を直接解析する方法と，コンピュータによって数値的に計算する方法がある．問題が複雑であり，数式を直接解くことが困難な場合には，シミュレーションを用いてモデルの挙動を調べることが有効である．

シミュレーションでは，現実の環境では容易に変更できない条件についても，仮想的に条件を変更して結果を比較できる．ただし，シミュレーションによる結果は，使用した数理モデル，データおよび仮定に基づくものであり，必ずしも現実と一致するとは限らない．そのため，得られた結果だけを確認するのではなく，モデルの前提や現実との対応関係を検討することが重要である．

## 2.4 本プロジェクトにおける数理モデリングの位置づけ

本プロジェクトでは，未来大学を発着するバスの混雑や遅延を対象としている．バスの運行方法を実際に変更し，複数の条件を試験することは，学生や一般利用者，バス事業者への影響が大きいため容易ではない．そこで，バスの運行と利用者の移動を数理モデルとして表現し，シミュレーション上で運行条件を変更することにより，それぞれの条件が混雑や遅延に与える影響を比較することを目指す．

数理モデリングを用いることで，単に感覚的に解決策の良し悪しを判断するのではなく，乗車人数，混雑度，乗車できない利用者数，遅延時間などの指標を用いて，運行方法の効果を定量的に評価できると考えられる．また，学生だけでなく，一般利用者やバス事業者への影響を含めて比較することで，運行条件によって生じる利益と不利益を整理することが可能となる．

ただし，中間発表時点では，本プロジェクトで使用する変数，仮定，数式およびアルゴリズムなどは決定しておらず，具体的な数理モデルの構築には至っていない．今後は，バスの運行や利用状況に関するデータを収集し，モデルに取り入れる要素と省略する要素を検討したうえで，目的に適した数理モデルを設計する必要がある．
