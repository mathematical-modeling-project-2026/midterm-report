# 第2章 数理モデリング

本章では，数理モデリングの概要と一般的な手順について述べる．また，本プロジェクトにおいて数理モデリングを用いる目的と，現在の活動における位置づけについて説明する．

## 2.1 数理モデリングとは

数理モデリングとは，実験や観察から得られた情報，および対象とする現象について設定した仮定に基づき，現実の現象や問題を数学的に表現することである［1］．このとき，現実の現象を数学的に表現したものを数理モデルという．数理モデルを用いることで，対象となる現象の性質を理解したり，条件を変更した場合の結果を予測したりすることが可能となる．

現実の現象には，多くの要素が複雑に関係しているため，そのすべてを数理モデルに取り入れることは困難である．そこで，対象とする問題において重要と考えられる要素を抽出し，それ以外の要素を省略することで，現象を扱いやすい形に単純化する．また，現実の状況を数学的に表現するために，必要に応じて仮定を設定し，変数や要素間の関係を数式や規則などによって記述する．

したがって，数理モデルは現実をそのまま再現したものではなく，特定の目的に応じて現実の一部を抽象化したものである［2］．同じ現象を対象とする場合でも，何を明らかにしたいかによって，モデルに取り入れる要素や設定する仮定は異なる．また，モデルから得られた結果は，使用したデータや仮定に依存するため，結果だけでなく，モデルの前提や適用できる範囲についても検討する必要がある．

## 2.2 数理モデリングの手順

数理モデルを構築するための唯一の手順が定められているわけではないが，一般的には，現実の問題を明確にし，仮定や変数を設定して数学的に表現した後，得られた結果を分析・評価するという流れで行われる［1］［2］．本プロジェクトでは，数理モデリングの手順を次のように整理する．

1. 対象とする現象や問題を明確にする  
2. 重要な要素を抽出し，必要な仮定を設定する  
3. 要素間の関係を数式や規則などによって表現する  
4. 構築した数理モデルを解析し，必要に応じてシミュレーションを行う  
5. 得られた結果を解釈し，現実の現象やデータと比較する  
6. 仮定やモデルの妥当性を評価し，必要に応じて修正する  

はじめに，現実の現象を観察し，何が問題となっているのか，何を明らかにしたいのかを整理する．次に，現象に関係する要素の中から，対象とする問題において重要と考えられるものを選択する．現実のすべての要素を扱うことは困難であるため，問題の本質を失わない範囲で単純化し，必要な仮定を設定する．

その後，選択した要素を変数として定め，要素間の関係を数式や規則などによって表現する．構築した数理モデルは，数学的に解析するほか，コンピュータによる数値計算やシミュレーションを用いて，条件を変更した場合の結果を調べる．

最後に，得られた結果が現実の現象に対してどのような意味を持つかを解釈し，観察結果や実測データと比較する．結果と現実の状況に大きな差がある場合には，設定した仮定，使用したデータ，モデルに取り入れた要素などを見直す必要がある．このように，数理モデリングは一度モデルを構築して終了するものではなく，結果の評価に基づいて前の段階へ戻り，モデルを修正する過程を繰り返すものである．

数理モデリングの基本的な流れを以下に示す．

> 現象・問題の設定  
> → 抽象化・仮定の設定  
> → 数学的な表現  
> → 解析・シミュレーション  
> → 結果の解釈・評価  
> → 仮定やモデルの見直し

## 2.3 本プロジェクトにおける数理モデリングの位置づけ

本プロジェクトでは，未来大学を発着する路線バスの混雑や遅延を対象としている．バスの運行方法を実際に変更し，複数の条件を繰り返し試すことは，日常的にバスを利用する学生や一般利用者，バス事業者に影響を与えるため容易ではない．そこで，バスの運行や利用者の移動を数理モデルとして表現し，仮想的に運行条件を変更することで，それぞれの条件が混雑や遅延に与える影響を比較することを目指す．

本プロジェクトにおける数理モデリングの役割は，現実のバス運行を完全に再現することではなく，解決策の効果を評価するために必要な要素を抽出し，異なる運行条件を同じ基準で比較できるようにすることである．これにより，感覚的に解決策の良し悪しを判断するのではなく，混雑度，遅延時間，乗車できない利用者数などの指標を用いて，運行方法による影響を定量的に評価することを目指す．

前期の中間発表時点では，対象とする問題を設定し，検討する解決策の方向性を決定した段階であり，具体的な数理モデルの構築や，そのモデルを用いたシミュレーションには着手していない．今後は，バスの運行や利用状況に関するデータを収集するとともに，モデルに取り入れる要素，設定する変数や仮定，評価に用いる指標などを検討する．そのうえで，本プロジェクトの目的に適した数理モデルを構築し，解決策の実現可能性と効果を検証する予定である．
