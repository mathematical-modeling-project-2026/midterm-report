# 第4章 過程・解決策

## 4.1 課題の難しさ

公立はこだて未来大学へ向かう路線バスは、授業開始前や授業終了後などの特定の時間帯に利用者が集中し、混雑が発生している。特に亀田支所前からはこだて未来大学までの区間では、満員で乗車できない学生や、混雑による遅延が発生することがあり、学生の通学に影響を与えている。 しかし、この問題は単純にバスの本数を増やせば解決できるものではない。増便を行うには、車両や運転手の確保が必要となり、バス会社の運行コストが増加する。また、混雑する時間帯や利用者数は、曜日、授業時間割、天候などによって変化するため、常に同じ運行方法が最適であるとは限らない。さらに、路線バスは学生だけでなく地域住民も利用している。そのため、学生の利便性だけを優先して運行方法を変更すると、途中の停留所を利用する地域住民の利便性が低下する可能性がある。つまり、 この課題では「学生の利便性」、「バス会社の運行効率」、「地域住民への影響」の3つを考慮する必要がある。このように、混雑を減らすことだけでなく、限られた車両や運転手の中で実現できるか、既存の利用者に大きな不利益がでないかを考慮しなければならない点が、本プロジェクトに置ける課題の難しさである。

## 4.2 やったこと・過程

本プロジェクトでは、公立はこだて未来大学行きのバスの混雑を緩和する方法について、グループで複数回の話し合いを行い、さまざまな解決策を検討した。 初めに、学生専用バスの運行、連節バスの導入、深夜バスの運行、快速バスの導入など、複数の案を挙げ、それぞれのメリットやデメリットについて議論した。学生専用バスは混雑を大きく緩和できる可能性がある一方で、新たな車両や運転手の確保が必要となるため、実現には多くの費用がかかると考えられた。また、連節バスについても輸送能力は高いが、車両の導入や道路環境などの課題があり、短期間で実施することは難しいと考えられた。 その中で、既存の運行ダイヤを大きく変更せずに実施できるという点から、快速バスの導入が最も現実的な解決策であるという結論に至った。快速バスであれば、一部の停留所を通過することで所要時間を短縮でき、利用者を通常便と分散させることによって混雑の緩和が期待できる。

## 4.3 解決策

本プロジェクトでは、亀田支所前からはこだて未来大学まで運行している既存のバスの一部を快速バスとして運行することを提案する。 快速バスは、利用者の少ない停留所を通過することで移動時間を短縮し、通常便と役割を分担することを目的としている。これにより、時間を優先したい学生は快速バスを利用し、途中の停留所を利用する乗客は通常便を利用するというように利用者が分散すると考えられる。 また、新たなバスを増便するのではなく、現在運行している便の一部を快速化するため、大きな設備投資を必要とせず、比較的実現しやすい点も特徴である。

## 4.4 解決策の良さ

快速バスを導入することによって、学生・バス会社・地域の三者に対して様々なメリットが期待できる。 学生にとっては、利用者が通常便と快速便に分散することで満員による乗り残しが減少し、通学時間の短縮や快適性の向上が期待できる。また、乗降にかかる時間が短縮されることで、バスの遅延も改善される可能性がある。 バス会社にとっては、新たな車両を導入することなく既存のダイヤを活用できるため、大きなコストをかけずにサービス改善を図ることができる。また、混雑が緩和されることで利用者満足度が向上し、公共交通の利用促進にもつながることが期待される。 さらに、利用者が快適に公共交通を利用できるようになれば、自家用車の利用抑制や交通渋滞の緩和など、地域全体にも良い影響を与える可能性がある。

## 4.6 数理モデリングで何するか

本プロジェクトでは、提案した快速バスの有効性を客観的に評価するため、数理モデリングを用いたシミュレーションを行う予定である。 本研究では、時間帯ごとの利用者数、停留所ごとの乗降人数、バスの運行本数、遅延時間などのデータを用いて、快速バス導入前後の状況を比較する。 シミュレーションでは、利用者の混雑状況や乗車率、所要時間、遅延時間などを評価し、快速バスを導入した場合にどの程度混雑が緩和されるのかを検証する。また、快速バスを運行する時間帯や運行本数を変更した場合についても比較を行い、より効果的な運行方法を検討する予定である。
