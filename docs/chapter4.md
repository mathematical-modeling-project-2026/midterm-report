# 第4章 過程・解決策

## 4.1 課題の難しさ

公立はこだて未来大学へ向かう路線バスは、授業開始前や授業終了後などの特定の時間帯に利用者が集中し、混雑が発生している。特に亀田支所から公立はこだて未来大学間では、満員で乗車できない学生や、混雑による遅延が発生することがあり、学生の通学に影響を与えている。 しかし、この問題は単純にバスの本数を増やせば解決できるものではない。増便を行うには車両や運転手の確保が必要となり、バス会社の運行コストが増加する。また、混雑する時間帯や利用者数は曜日や授業時間割によって変化するため、常に同じ運行方法が最適であるとは限らない。さらに、路線バスは学生だけでなく地域住民も利用しているため、学生の利便性だけを優先した運行に変更することは難しい。 このように、「学生の利便性向上」と「バス会社の運行効率」、「地域住民への影響」の3つを考慮しながら解決策を検討する必要があることが、本プロジェクトの課題の難しさである。そのため、本プロジェクトでは実現可能性の高い解決策を検討し、数理モデリングによってその有効性を評価することを目指す。 このような複数の条件を同時に考慮する必要があるため、経験や直感だけで最適な運行方法を判断することは難しい。そこで、本プロジェクトでは数理モデリングを用いてバスの運行をシミュレーションし、混雑緩和と運行効率を両立できる運行方法を検討する。
